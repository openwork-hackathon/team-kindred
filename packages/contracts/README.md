# @kindred/contracts

Smart contracts for Kindred - built with Foundry.

## Contracts

- `KindredReputation.sol` - Reputation tracking contract
- `KindredReview.sol` - Review NFT contract  
- `KindredHook.sol` - Uniswap v4 Hook for reputation-based protection

## Getting Started

```bash
# Build
forge build

# Test
forge test

# Deploy
forge script script/Deploy.s.sol --rpc-url base --broadcast
```
