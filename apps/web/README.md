# @kindred/web

Frontend application for Kindred - built with Next.js 14.

## Getting Started

```bash
pnpm dev
```

## Tech Stack

- Next.js 14 (App Router)
- TailwindCSS
- RainbowKit + wagmi (wallet connection)
- viem (Ethereum interactions)

## Features

- [ ] Landing page
- [ ] Wallet connection
- [ ] Review submission form
- [ ] Review feed
- [ ] Reputation display
- [ ] Staking interface
